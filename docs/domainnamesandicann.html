In the summer of 1994, Jim Clark and his young web browser pioneer launched the fastest growing software company in the world. Bill Gates offered to work with Netscape together and even offered to buy it. But Netscape didn’t accept this order. Gates made a smart move that he sent out a memo to all Microsoft employees titled the internet tidal wave. This move made Microsoft declare themselves as the winner of the Browser Wars.

Another essential law was Metcalfe’s Law. Metcalfe’s Law stated that every new node (a device such as a computer) that is added to the internet increases the internet’s value by a steep curve. It is the Moore’s Law of connectivity. For example: If there are 2 users in a network, there are 2 connections between a network; if there are 3 users in a network, there are 6 connections between a network. If there are 100 users in a network, then there are 5000 connections between a network. As the number of users increases, the usefulness of the internet increases. Both laws worked together in the late 1990’s to set the stage for a new financial bubble.
